What is Git?

Git is a version control system used to track changes in code during software development, allowing developers to collaborate with each other while efficiently managing code changes. A history of the modifications is managed and it allows to revert the current code to a previous state more easily.

Git Functions Implemented

  • Init - Initializes a new Git repository
  • Add - Stages changes for the next commit
  • Commit - Saves your changes locally
  • Push - Sends local commits to the remote repository
  • Pull - Updates your local repo from the remote repo
  • Branch - Manages multiple lines of development
  • Merge - Combines branches together
  • Checkout - Switches branches or restores files
  • Log - Show commit logs
  • Rm - Remove files from the working tree and from the index
  • Status - Displays the state of the working directory
  • Reset - Reset current HEAD file to the specified state
